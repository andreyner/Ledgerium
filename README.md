Тестовое задание T4 для Юнистрим банка. C#

Напишите Web API, позволяющее зачислять и списывать средства у
указанного клиента, отменять указанную транзакцию и получать актуальный баланс клиента.

Чтобы развернуть сервисы нужно выполнить команду docker-compose build, а затем docker-compose up -d
Далее можно использовать Swagger http://localhost:15542/swagger/index.html
Для просмотра логов подключен SEQ http://localhost:15341/#/events?range=1d
